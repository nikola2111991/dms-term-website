# DMS-Term Website

Profesionalni website za firmu DMS-Term iz Novog Sada - hidroizolacija i limeni krovovi.

## O firmi

- **Firma:** DMS-Term
- **Vlasnik:** Dejan Masic
- **Lokacija:** Veternicka 13, 21000 Novi Sad
- **Delatnost:** Hidroizolacija, limeni krovovi, ravni krovovi
- **Osnovana:** 2014. godine

## Tehnologije

- HTML5 / CSS3 / JavaScript
- Responsive design (mobile-first)
- Google Fonts (Poppins)
- Font Awesome ikone

## Funkcionalnosti

- Mobilni meni (hamburger)
- Smooth scroll navigacija
- Galerija sa lightbox-om
- Testimonial slider
- FAQ accordion
- Kontakt forma sa Viber integracijom
- Counter animacije
- Scroll reveal animacije
- Floating poziv/Viber dugmici
